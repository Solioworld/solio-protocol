## getFactory()

```ts copy
function getFactory(client, props): Promise<`0x${string}`>
```

Fetches the factory address from the router contract.

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `client` | [`ReadClient`](../../types.mdx#ReadClient) | [ReadClient](../../types.mdx#ReadClient) The read client to use. |
| `props` | [`RouterGetFactoryProps`](get-factory.mdx#RouterGetFactoryProps) | [RouterGetFactoryProps](get-factory.mdx#RouterGetFactoryProps) The properties for the function. |

<a id="RouterGetFactoryProps"></a>

```ts copy
type RouterGetFactoryProps: {
  address: Address;
};
```

#### Returns

`Promise`&lt;\`0x$\{string\}\`&gt;

A promise that resolves to the factory address.

### Example

```ts copy filename="router.ts"
import { router } from '@solio/core';
import { createClient, http } from 'viem';
import { sepolia } from 'viem/chains';
const readClient = createClient({
 chain: sepolia,
 transport: http(),
})
const factoryAddress = await router.read.getFactory(readClient, {
 address: '0x...' // router contract address
});
console.log(factoryAddress);
```
