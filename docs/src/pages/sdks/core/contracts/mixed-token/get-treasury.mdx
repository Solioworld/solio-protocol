## getTreasury()

```ts copy
function getTreasury(client, props): Promise<`0x${string}`>
```

Reads the project treasury address of a mixed token.

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `client` | [`ReadClient`](../../types.mdx#ReadClient) | [ReadClient](../../types.mdx#ReadClient) The read client to use. |
| `props` | [`MixedTokenGetTreasuryProps`](get-treasury.mdx#MixedTokenGetTreasuryProps) | [MixedTokenGetTreasuryProps](get-treasury.mdx#MixedTokenGetTreasuryProps) The properties to use. |

<a id="MixedTokenGetTreasuryProps"></a>

```ts copy
type MixedTokenGetTreasuryProps: {
  address: Address;
};
```

#### Returns

`Promise`&lt;\`0x$\{string\}\`&gt;

The project treasury address.

### Example

```ts copy filename="mixed-token.ts"
import { mixedToken } from '@solio/core';
import { createClient, http } from 'viem';
import { sepolia } from 'viem/chains';
const readClient = createClient({
 chain: sepolia,
 transport: http(),
});
const treasury = await mixedToken.read.getTreasury(readClient, {
 address: '0x...' // mixed token address
});
```
