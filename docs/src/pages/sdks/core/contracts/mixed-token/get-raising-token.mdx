## getRaisingToken()

```ts copy
function getRaisingToken(client, props): Promise<`0x${string}`>
```

Reads the raising token address of a mixed token.

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `client` | [`ReadClient`](../../types.mdx#ReadClient) | [ReadClient](../../types.mdx#ReadClient) The read client to use. |
| `props` | [`MixedTokenGetRaisingTokenProps`](get-raising-token.mdx#MixedTokenGetRaisingTokenProps) | [MixedTokenGetRaisingTokenProps](get-raising-token.mdx#MixedTokenGetRaisingTokenProps) The properties to use. |

<a id="MixedTokenGetRaisingTokenProps"></a>

```ts copy
type MixedTokenGetRaisingTokenProps: {
  address: Address;
};
```

#### Returns

`Promise`&lt;\`0x$\{string\}\`&gt;

The raising token address.

### Example

```ts copy filename="mixed-token.ts"
import { mixedToken } from '@solio/core';
import { createClient, http } from 'viem';
import { sepolia } from 'viem/chains';
const readClient = createClient({
 chain: sepolia,
 transport: http(),
});
const raisingTokenAddress = await  mixedToken.read.getRaisingToken(readClient, {
 address: '0x...'  // mixed token address
});
console.log(raisingTokenAddress);
```
