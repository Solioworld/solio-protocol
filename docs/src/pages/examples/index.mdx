# Examples

This section provides a collection of examples to help you get started with building your own applications using Solio Protocol.

## Quick Start

Get started with Solio Protocol by following these simple steps:

### Clone the repository

```sh copy
git clone https://github.com/Solioworld/solio-protocol
```

### Cd into the directory

```sh copy
cd solio-protocol/examples
```

### Explore the examples

We have several examples to help you get started:

- [Bonding Curve Calculator](/examples/bonding-curve-calculator)
- [React](/examples/react)
- [React-Native](/examples/react-native)
- [Expo](/examples/expo)
- [Vue](/examples/vue)
