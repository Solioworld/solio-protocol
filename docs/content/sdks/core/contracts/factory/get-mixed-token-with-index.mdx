## getMixedTokenWithIndex()

```ts copy
function getMixedTokenWithIndex(client, props): Promise<`0x${string}`>
```

Reads the mixed token address with index.

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `client` | [`ReadClient`](../../types.mdx#ReadClient) | [ReadClient](../../types.mdx#ReadClient) The read client to use. |
| `props` | [`FactoryGetMixedTokenAddressWithIndexProps`](get-mixed-token-with-index.mdx#FactoryGetMixedTokenAddressWithIndexProps) | [FactoryGetMixedTokenAddressWithIndexProps](get-mixed-token-with-index.mdx#FactoryGetMixedTokenAddressWithIndexProps) The properties to use. |

<a id="FactoryGetMixedTokenAddressWithIndexProps"></a>

```ts copy
type FactoryGetMixedTokenAddressWithIndexProps: {
  address: Address;
  mixedTokenIndex: bigint;
};
```

#### Returns

`Promise`&lt;\`0x$\{string\}\`&gt;

The mixed token address.

### Example

```ts copy filename="factory.ts"
// for reading the index 0 mixed token
import { factory } from '@solio/core';
import { createClient, http } from 'viem';
import { sepolia } from 'viem/chains';
const readClient = createClient({
 chain: sepolia,
 transport: http(),
});
const mixedTokenAddress = await factory.read.getMixedTokenWithIndex(readClient, {
 address: '0x...', // factory contract address
 mixedTokenIndex: 0n
});
console.log(mixedTokenAddress);
```
