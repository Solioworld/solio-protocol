## getEstimateBurn()

```ts copy
function getEstimateBurn(client, props): Promise<{
  raisingAmount: bigint;
  platformFee: bigint;
  projectFee: bigint;
 }>
```

Estimates the native token amount that needs to be burned when a certain amount of mixed token is burned.

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `client` | [`ReadClient`](../../types.mdx#ReadClient) | [ReadClient](../../types.mdx#ReadClient) The read client to use. |
| `props` | [`MixedTokenGetEstimateBurnProps`](get-estimate-burn.mdx#MixedTokenGetEstimateBurnProps) | [MixedTokenGetEstimateBurnProps](get-estimate-burn.mdx#MixedTokenGetEstimateBurnProps) The properties to use. |

<a id="MixedTokenGetEstimateBurnProps"></a>

```ts copy
type MixedTokenGetEstimateBurnProps: {
  address: Address;
  mixedTokenAmount: bigint;
};
```

#### Returns

`Promise`&lt;\{
  `raisingAmount`: `bigint`;
  `platformFee`: `bigint`;
  `projectFee`: `bigint`;
 \}&gt;

The native token amount, platform fee, and project fee.

### Example

```ts copy filename="mixed-token.ts"
import { mixedToken } from '@solio/core';
import { createClient, http } from 'viem';
import { sepolia } from 'viem/chains';
const readClient = createClient({
 chain: sepolia,
 transport: http(),
});
const { raisingAmount, platformFee, projectFee } = await mixedToken.read.getEstimateBurn(readClient, {
 address: '0x...', // mixed token address
 mixedTokenAmount: 1n
});
console.log(raisingAmount, platformFee, projectFee);
```
